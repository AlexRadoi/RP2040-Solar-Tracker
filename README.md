# RP2040-Solar-Tracker
RP2040 Single axis solar tracker
This project was my final masters project and involves the development of a simple single axis solar tracker based on an open loop astronomical algorithm and providing output power through a SEPIC converter and logging all data by creating an .cvs file on an SD card.
